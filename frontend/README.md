# Frontend

Веб-клиент для `Kasay Bank` на `React + Vite`.

## Запуск

```bash
cd frontend
npm install
npm run dev
```

По умолчанию dev-сервер проксирует запросы `/api/*` на `http://localhost:8080`.

Если нужен другой backend URL, создайте `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

## Что реализовано

- регистрация
- логин
- обновление токенов
- проверка авторизации
- просмотр своих карт
- список получателей с картами
- выпуск виртуальной карты
- пополнение счета
- перевод по номеру карты
- перевод по телефону

## Интерфейс

- отдельный экран входа и регистрации
- минималистичный кабинет без лишних технических пояснений
- отдельные разделы для карт, переводов и безопасности
- зелёное подтверждение после успешного перевода
